# 로또 게임

---

- Number
  - 숫자만 입력할 수 있다.

- Ball
  - 1~45까지만 입력할 수 있다.
  - 음수는 입력할 수 없다.
- Lotto
  - 6개의 볼을 저장 할 수 있다.
- WinningLotto
  - 1등 당첨번호를 저장한다.
- LottoGame
  - 구매한 로또 번호를 입력받는다. 
  - 로또 결과를 출력한다.
- LottoMachine
  - 로또 번호를 랜덤으로 생성한다.
  - 1등 당첨번호를 생성한다.
- Seller
  - 로또 금액을 입력받으면 로또 갯수를 출력한다.
